# ReguVigil: End-to-End Automated Pharmacovigilance

ReguVigil is a robust, multi-agent AI pipeline built for continuous trial monitoring and automated regulatory compliance. It digests complex, unstructured FDA guidelines and seamlessly triggers global clinical trial protocol updates.

## 🚀 One-Command Launch (Demo Mode)

The entire application stack (PostgreSQL, FastAPI Backend, React Frontend, and Python Scheduler) has been fully Dockerized. 

Ensure you have Docker and Docker Compose installed.

1. **Clone and Enter Directory**:
   ```bash
   cd reguvigil
   ```

2. **Launch the Stack**:
   ```bash
   docker-compose up --build
   ```

3. **Access the Application**:
   Open your browser to [http://localhost:3000](http://localhost:3000)

## 🎭 Demo Showcase Guide

To effectively showcase the ReguVigil workflow during your presentation, follow this script:

### Step 1: The Regulatory Command Center
- **Login As**: `Priya` (Regulatory Affairs)
- **Action**: Go to `/dashboard/regulatory`. Note the "Active Rule v1.2" parameters.
- **Demo**: Click **Upload Guideline**. Choose the synthetic `FDA_Cardiac_Guidance_2026.pdf` provided in the root directory.
- **Result**: You will immediately see the **Pipeline Live View** animate as the 4 Agents execute. Within ~10 seconds, the dashboard will flash a red Alert Banner indicating a rule update (v1.3) and show the side-by-side JSON diff!

### Step 2: Global Data Manager Impact
- **Login As**: `Arjun` (Data Manager)
- **Action**: Go to `/dashboard/datamanager`. 
- **Result**: Show the global 10-site breakdown. Demonstrate the immediate real-time effect of the new v1.3 rule: 3 specific patients have now crossed the restricted 28ms HRV threshold and are prominently flagged in red. Click the "Export CSV" button to demonstrate regulatory reporting data extraction.

### Step 3: Site-Level Principal Investigator View
- **Login As**: `Ramesh` (Doctor/PI for Site 3)
- **Action**: Go to `/dashboard/doctor`.
- **Result**: Dr. Ramesh ONLY sees patients from Apollo Chennai (Site 3). Show how the interactive Recharts visualization directly highlights the threshold breach for his specific patients over the 30-day timeline.

### Step 4: The Final Safety Report
- **Action**: From the Regulatory Dashboard or Pipeline View, click on the generated **PV Safety Report**.
- **Result**: Showcase the pristine, physical document-styled A4 report view that was automatically generated by Agent 4. Click **Download PDF** to show the live PDF streaming from the backend.
